## Avoir les informations d'un challenge en particulier

### Requête

| URL                 | Method |
|:--------------------| :--- |
| /api/challenge/{id} | GET |

- **id** : id du challenge

### Réponse

#### Succès

- code : **200**
- contenu : 
```json
{
    "id": 1,
    "name": "Ecocup",
    "points": 1,
    "description": "\ud83e\udd64R\u00e9cup\u00e9rer une \u00e9cocup"
}
```

#### Erreurs

- code : **404**
- contenu : 
```json
{
    "message": "No challenge found"
}
```

|:---------------------------| :--- |
